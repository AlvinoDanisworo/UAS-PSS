RESTful API dengan Token Authentication untuk Learning Management System

üìå Authentication

API menggunakan JWT (JSON Web Token) authentication. Untuk mengakses protected endpoints, Anda perlu mendapatkan access token terlebih dahulu.

POST /api/v1/login

Login dan dapatkan JWT access & refresh tokens

{
  "username": "admin",
  "password": "admin"
}

Response:

{
  "access": "eyJ0eXAiOiJKV1QiLCJhbGc...",
  "refresh": "eyJ0eXAiOiJKV1QiLCJhbGc...",
  "user": {
    "id": 1,
    "username": "admin",
    "first_name": "Admin",
    "last_name": "User",
    "email": "admin@lms.com"
  },
  "message": "Login berhasil"
}
POST /api/v1/refresh

Refresh access token menggunakan refresh token

{
  "refresh": "eyJ0eXAiOiJKV1QiLCJhbGc..."
}

Response:

{
  "access": "eyJ0eXAiOiJKV1QiLCJhbGc..."
}

{
  "username": "newuser123",
  "password": "password123",
  "email": "user@example.com",
  "first_name": "John",
  "last_name": "Doe"
